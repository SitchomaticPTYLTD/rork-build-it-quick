# Add three new text tools

Add three new script functions to the main screen:

- **Remove all spaces** — strips every space character from the text.
- **Convert spaces / chosen symbol to new lines** — replaces spaces (or a custom symbol the user types in) with line breaks, so each chunk ends up on its own line.
- **Remove non-alphanumeric prefix** — for every line, trims any leading characters that aren't letters or numbers (e.g. `• hello` → `hello`, `-- test` → `test`).

Each appears as its own card on the main screen alongside the existing tools, with the same look, feel, and input/output flow.
